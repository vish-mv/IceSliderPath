Java Algorithmic program for find the shortest path in a ice slider map.

There are 3 Java files (classes) in the project :-

IceSlider is the main class with the main method.
MapParser is the class is the one that reads txt file and identify the data.
ShortestPath is the class that contains algorithm to find the shortest path.
